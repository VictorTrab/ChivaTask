# Design Tokens

## Color

- Fondo claro: `#F5F7FA`
- Superficie: `#FFFFFF`
- Superficie suave: `#F8FAFC`
- Borde: `#D8E2EA`
- Texto primario: `#102033`
- Texto secundario: `#64748B`
- Marca/sidebar: `#123F35`
- Accion primaria: `#16775F`
- Hover primario: `#0F5F4A`
- Warning/vencida: `#D97706`
- Error: `#DC2626`
- Info: `#2563EB`

## Forma y ritmo

- Radio boton: 10px
- Radio card: 12-14px
- Inputs: altura estable minima 40-44px
- Spacing: escala de 4/8px; evitar valores arbitrarios por pantalla.
- Tipografia: Segoe UI, Arial fallback; titulos 17-18px, card title 14-15px, body 13px, secundario 12px.

## Estados

Definir variantes para hover, pressed, disabled, focus, selected, loading, empty, filtered-empty, offline-cache, recoverable-error y blocking-error.
