# Visual Scorecard

Puntuar cada categoria como `OK`, `Riesgo` o `Falla`.

## Categorias

- Identidad: pantallas coherentes con ChivaTask academico/productivo.
- Jerarquia: titulo, acciones, estados y contenido escaneables.
- Componentes: reutilizacion real, sin duplicacion innecesaria.
- Responsive: 960x640 usable, 1366x768 sin recortes, 1920x1080 equilibrado.
- Accesibilidad: foco, teclado, contraste, nombres accesibles.
- Estados: loading, empty, filtered-empty, offline-cache, recoverable-error, blocking-error.
- Modo oscuro: no invertir sin criterio; contraste y bordes revisados.
- Texto: espanol correcto, mensajes accionables, sin tecnicismos como mensaje principal.
- Evidencia: tests, compileall, diff check y capturas/smoke.

Si alguna categoria queda en `Falla`, no aprobar el cierre.
